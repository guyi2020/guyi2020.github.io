## 1.下载安装

 `npm install -g n`

##  2.下载指定版本(后面跟版本号)

 `sudo n 14.15.0`

##  3.下载最新版本

 `sudo n latest`

##  4.显示已安装哪些版本

 `n ls`

##  5.切换使用版本(后面跟版本号)

 `sudo n 14.15.0`
 也可以输入
 `sudo n`
 查看已安装版本，上下切换使用哪个版本

##  6.删除指定版本

 `sudo n rm 14.15.0`

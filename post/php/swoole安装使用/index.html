Swoole安装使用

2021-06-22
技术
约 141 字 预计阅读 1 分钟

文章目录

    • 下载Swoole源码
    • 解压并进入到目录
    • 执行phpize
    • 编译
    • 添加swoole.so扩展，并重启

  1. 下载Swoole源码

    https://github.com/swoole/swoole-src/releases

  2. 解压并进入到目录

    cd swoole-src-4.6.7

  3. 执行phpize

    /Applications/MAMP/bin/php/php7.2.8/bin/phpize

    注意：PHP版本，用的是7.2以上

  4. 编译

    ./configure --with-php-config=/Applications/MAMP/bin/php/php7.2.8/bin/php-config

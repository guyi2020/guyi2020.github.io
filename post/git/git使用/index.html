Git使用

2021-06-11
技术
约 170 字 预计阅读 1 分钟

文章目录

    • 新建远程分支
    • 新建本地分支
    • 删除本地分支
    • 目标分支有冲突不能合并到master问题

  1. 新建远程分支

    git push --set-upstream origin dev

    注意：dev 是远程分支的名称

  2. 新建本地分支

    git checkout -b dev

  3. 删除本地分支

    git branch -d dev

  4. 目标分支有冲突不能合并到master问题

    • 首先切换到master分支 git checkout master
    • 将master的更新pull到本地 git pull origin master
    • 切换到目标分支 git checkout 目标分支名
    • 将master合并到目标分支 git merge master
    • 合并可能会出现冲突，处理完成后，master的更改已经成功合并到目标分支中
